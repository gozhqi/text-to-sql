# 系统架构设计

```
┌─────────────────────────────────────────────────────────────────────┐
│                        用户交互层                                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐                  │
│  │  Web UI     │  │  API接口    │  │  第三方集成 │                  │
│  └─────────────┘  └─────────────┘  └─────────────┘                  │
└─────────────────────────────────────────────────────────────────────┘
                                 ↓
┌─────────────────────────────────────────────────────────────────────┐
│                      对话管理层 (Context Manager)                    │
│  • 会话ID管理        • 多轮上下文存储        • 意图识别              │

                                 ↓
┌─────────────────────────────────────────────────────────────────────┐

```

## 核心模块说明

### 1. Schema Retriever (Schema检索器)

**职责**: 从数据库Schema中找到与用户问题相关的表

**实现方式**:
- 向量检索: 使用Sentence Transformers将Schema描述向量化
- 关键词匹配: 检查问题中是否包含表名/字段名
- 混合检索: 结合两种方式提高召回率

**关键代码**: `app/core/schema_retriever.py`

### 2. Context Manager (上下文管理器)

**职责**: 管理多轮对话的上下文

**功能**:
- 会话创建与维护
- 历史对话存储
- 问题改写
- 意图识别

**关键代码**: `app/core/context_manager.py`

### 3. SQL Generator (SQL生成器)

**职责**: 调用LLM生成SQL

**功能**:
- Prompt构建
- Few-shot示例注入
- LLM调用
- 结果解析

**关键代码**: `app/core/sql_generator.py`

### 4. SQL Validator (SQL校验器)

**职责**: 确保生成的SQL安全可执行

**功能**:
- 危险操作拦截
- SQL注入检测
- 语法校验
- 表名校验

**关键代码**: `app/core/sql_validator.py`

## 数据流

```
用户问题
    ↓
[Context Manager] 检查是否多轮对话
    ↓ (问题改写)
[Schema Retriever] 检索相关表
    ↓ (相关Schema)
[SQL Generator] 构建Prompt + 调用LLM
    ↓ (生成SQL)
[SQL Validator] 安全校验
    ↓ (合法SQL)
执行SQL → 返回结果
```

## 技术选型理由

| 技术 | 选型理由 |
|------|----------|
| FastAPI | 高性能异步框架，自动生成API文档 |
| ChromaDB | 轻量级向量数据库，易于本地部署 |
| Sentence Transformers | 多语言支持，中文效果好 |
| OpenAI GPT-4 | SQL生成能力强，中文理解好 |

## 扩展性设计

### 支持多种LLM
```python
# 通过配置切换LLM提供商
LLM_PROVIDER=openai  # 或 azure, anthropic, local
```

### 支持多种数据库
```python
# 通过配置切换数据库类型
DB_TYPE=mysql  # 或 postgresql, sqlite
```

### 支持自定义Prompt
```python
# 可以覆盖默认的Prompt模板
class CustomPromptBuilder(SQLPromptBuilder):
    SYSTEM_PROMPT = "你的自定义提示词..."
```
